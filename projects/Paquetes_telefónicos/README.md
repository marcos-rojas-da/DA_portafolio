# Análisis de Paquetes Telefónicos

## Objetivo
El objetivo fue analizar el comportamiento de los usuarios para identificar patrones
de consumo y posibles oportunidades de mejora en los planes.

## ¿Qué hice?
- Limpieza y exploración de datos
- Análisis de variables clave
- Visualización de patrones de uso
- Gráficas
- Pruebas de hipótesis

## Herramientas
- Python (pandas, matplotlib, scipy)
- Jupyter Notebook

## Principales insights
- Existen diferencias marcadas en el comportamiento de consumo entre los tipos de plan, lo que permite identificar perfiles claros de usuarios según su uso de minutos, mensajes y datos.
- Una proporción relevante de clientes excede regularmente los límites de su plan, generando cargos adicionales y señalando una posible desalineación entre el plan contratado y el uso real.
- Los planes de menor costo concentran mayor variabilidad en el consumo, mientras que los planes premium muestran patrones más estables y predecibles.
- Estos hallazgos pueden apoyar estrategias de retención, optimización de planes y recomendaciones personalizadas para mejorar la experiencia del cliente.
